Программа имеет два способа вывода информации:
1. В отдельный файл
2. В консоль

Чтобы воспользоваться первым вариантом, вам необходимо
в аргументах программы указать названия двух файлов через пробел.
Файлы должны называться input.txt и output.txt и должны находиться в корневой папке программы.
В первом файле вам необходимо будет указать название требуемой вам операции с числами и сами числа.

Программа имеет три вида операций с числами:
1)умножение всех введенных чисел - mul
2)сложение всех введенных чисел - add
3)умножение первых двух введенных чисел и сложение с остальными введенными числами - muladd

После названия операции можно вводить только числа.
Информацию в первый файл необходимо вводить в следующем формате:
Пример - add 1 2 3 4 5

Когда вы вели данные в первый файл запустите программу. Ответ будет во втором файле.


Чтобы воспользоваться вторым вариантом ввода информации, вам необходимо
в аргументах программы указать - - через пробел.

Далее запустите программу.
Программа попросит вас ввести в консоль название требуемой вам операции и ваши числа.

Программа имеет три вида операций с числами:
1)умножение всех введенных чисел - mul
2)сложение всех введенных чисел - add
3)умножение первых двух введенных чисел и сложение с остальными введенными числами - muladd

После названия операции можно вводить только числа.
Информацию в первый файл необходимо вводить в следующем формате:
Пример - add 1 2 3 4 5

Когда вы вели данные в консоль нажмите enter. Ответ будет выведен в консоль.
